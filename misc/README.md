# Misc & Design Archive

This directory contains design reference materials, mockups, stitch code exports, and legacy root files created during initial development iterations.

## Directory Contents

### 🎨 `design_mocks/`
Contains original UI design mockups, screenshots, and prototype exports from Stitch:
- **`rentcan_payments_ledger_final/`**: Ledger & transaction table UI design mockups and screenshots.
- **`rentcan_premium_landing_page_final/`**: Early landing page prototype code and screenshots.
- **`rentcan_premium_swiss/`**: Swiss architectural design tokens (`DESIGN.md`).
- **`rentcan_property_dashboard_final/`**: Property management dashboard UI design mocks.
- **`rentcan_refined_swiss_landing_page/`**: Refined landing page layout code and screens.
- **`rentcan_reports_vault_final/`**: Vault and document storage UI mockups.
- **`rentcan_unified_secure_login_final/`**: Original login screen mockup.
- **`tricity_trust_system/`**: Trust system design specs (`DESIGN.md`).

### 📦 `legacy_root_files/`
Contains initial single-file prototypes (`index.html`, `dashboard.html`) kept for historical reference.

> **Note**: Active production code is served exclusively from the `public/` directory.
